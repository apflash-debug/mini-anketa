\# Мини-анкета



Простое full-stack приложение — анкета с backend на FastAPI и frontend на HTML+JS.



\## Технологии

\- Backend: Python, FastAPI, Uvicorn

\- Frontend: HTML + JavaScript



\## Установка и запуск



\### 1. Установи зависимости

pip install fastapi uvicorn



\### 2. Запусти backend



uvicorn main:app --reload



\### 3. Открой frontend

Открой файл `index.html` в браузере.



\## API

\- `GET /questions` — возвращает список вопросов

\- `POST /answers` — принимает ответы пользователя
